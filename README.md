# Название проекта: Интерактивный дашборд проектов с настраиваемыми виджетами (Angular)

# Описание:
Проект представляет собой SPA (Single Page Application) на базе Angular, 
предназначенное для агрегации и визуализации данных о ходе выполнения проектов. 

В основе архитектуры лежит компонентный подход (component-based architecture), 
что позволило разделить пользовательский интерфейс на небольшие и изолированные компоненты.

# Технический стек и реализация:

Фреймворк: Angular (версия 20).

Архитектура: Приложение структурировано с использованием компонентов, 
сервисов, модулей и маршрутизации.

Получение данных: Интеграция с localStorage для получения и сохранения информации о проектах через сервисы Angular.
